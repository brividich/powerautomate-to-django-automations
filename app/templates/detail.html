Come Usare Il Pairing
1. Collega la tabella target Trascina una colonna dal pannello di sinistra nel riquadro Pairing Tabella Target quando vuoi salvare il valore del flow in una colonna reale del database.
2. Collega il runtime del portale Trascina un campo logico nel riquadro Pairing Runtime Portale. Questo mapping serve al portale Django e puo' essere diverso dalla colonna SQL.
3. Salva anche se non e' completo Solo target: package valido con warning. Solo runtime: package valido. Entrambi: scenario completo consigliato per l'import nel portale.
Esempio Rapido
Lo stesso campo Power Automate puo' puntare a due destinazioni diverse. 
Campo flow: EmailDipendente
Target table: request_email
Runtime portale: dipendente_email

Salvataggio finale:
- approved_target_field_mapping["EmailDipendente"].target_field = "request_email"
- approved_runtime_field_mapping["EmailDipendente"].target_field = "dipendente_email"
